"A step-by-step learning path for analyzing patient vital signs using Python, from basic descriptive statistics to advanced data visualization."

# 🏥 Patient Vital Signs Analysis: A Learning Path

This repository contains a series of Jupyter Notebooks designed to teach healthcare data analysis using Python, Pandas, and Matplotlib.

## 📂 Repository Structure

### 1. Beginner Level: Starting with Python
* **File:** `simple_beginner_vitals_notebook.ipynb`
* **Concepts:** Importing libraries, creating dataframes, and basic plotting.

### 2. Intermediate Level: Handling Complexity
* **File:** `next_level_vitals_notebook.ipynb`
* **Concepts:** Data filtering, handling missing values (imputation), and group comparisons.

### 3. Deep Dive: Descriptive Statistics & Correlation
* **File:** `Patient_symptom_analysis_by_descriptive_statististics.ipynb`
* **Concepts:** Measuring variation (IQR, Std Dev), understanding distributions, and Pearson correlation (Heart Rate vs. Temp).

## 🛠️ Requirements
* Python 3.x
* Pandas
* Numpy
* Matplotlib / Seaborn
